Ave is an OAuth 2.0 + OpenID Connect provider with privacy-first identity, optional end-to-end encryption key delivery, and delegated app-to-app access.

## What you can build

- User sign-in via Authorization Code flow
- PKCE login for web/mobile public clients
- Server-side token exchange for confidential clients
- OIDC profile retrieval (`/userinfo`) and ID token validation
- Connector app-to-app delegation (`token-exchange` grant)
- Signature workflows using identity-backed Ed25519 keys
- Optional per-app encryption key handoff for E2EE apps

## Important URLs (don’t mix these)

- User-facing auth UI: `https://aveid.net/signin`
- Connector UI: `https://aveid.net/connect`
- API base: `https://api.aveid.net`
- OIDC discovery: `https://api.aveid.net/.well-known/openid-configuration`
- JWKS: `https://api.aveid.net/.well-known/jwks.json`

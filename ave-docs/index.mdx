Ave is an OAuth 2.0 + OpenID Connect provider. It handles identity, authentication, token issuance, delegated access, cryptographic signing, and optional per-app encryption key delivery.

## Important URLs

<Warning>
  Ave uses two separate hosts. The sign-in UI lives on `aveid.net`. All API calls go to `api.aveid.net`. Using the wrong host is the most common integration mistake.
</Warning>

| Purpose | URL |
|---|---|
| Sign-in UI | `https://aveid.net/signin` |
| Connector consent UI | `https://aveid.net/connect` |
| OAuth token endpoint | `https://api.aveid.net/api/oauth/token` |
| Userinfo endpoint | `https://api.aveid.net/api/oauth/userinfo` |
| OIDC discovery | `https://api.aveid.net/.well-known/openid-configuration` |
| JWKS | `https://api.aveid.net/.well-known/jwks.json` |
